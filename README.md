# SecPass 🦆

Менеджер паролей с шифрованием AES-256-CBC/ChaCha20, защитой памяти и графическим интерфейсом.

## Возможности

- **Шифрование**: AES-256-CBC + ChaCha20, ключи выводятся через Argon2id
- **Двухслойное шифрование**: тела записей шифруются дважды
- **Защита памяти**: SecureString с mlock/VirtualLock (без свопинга на диск)
- **Ключ-файл**: опциональный второй фактор при разблокировке
- **Группы**: древовидная структура, перетаскивание записей
- **Генератор паролей**: со встроенным расчётом энтропии
- **Вкладки**: работа с несколькими хранилищами одновременно
- **Поиск**: фильтрация записей по имени и URL

## Криптография

```
Пароль (+ ключ-файл)
       │
       ▼
   Argon2id ──> derived_key
       │
       ▼
master_key (32 сл. байта) → AES-256-CBC → файлы записей/групп
```

Тела записей шифруются дважды (внутренний + внешний слой AES-256-CBC).

## Безопасность

- Минимальная энтропия пароля: **80 бит**
- Пароли в памяти защищены `VirtualLock`/`mlock`
- При блокировке хранилища данные зануляются (`zeroize`)
- Имена файлов — SHA256(UUID), без утечки содержимого
